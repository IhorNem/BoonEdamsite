# Сайт Parkan.ua - Офіційний дилер Boon Edam в Україні

## Опис проєкту
Повнофункціональний веб-сайт для офіційного дилера революційних дверей Boon Edam в Україні - компанії Parkan.ua.

## Структура проєкту

```
parkan-revolving-doors/
├── index.html              # Головна сторінка
├── README.md               # Документація проєкту
├── css/
│   ├── style.css          # Основні стилі сайту
│   └── product.css        # Стилі для сторінок продуктів
├── js/
│   └── script.js          # JavaScript функціонал
├── images/                 # Зображення продукції
│   └── placeholder.jpg    # Placeholder для зображень
└── products/              # Сторінки окремих продуктів
    ├── tqa.html           # TQA - Автоматичні двері
    ├── tournex.html       # Tournex - Високопродуктивні
    ├── duotour.html       # Duotour - 3 в 1
    ├── boonassist-tq.html # BoonAssist TQ - З підсиленням
    ├── tqm.html           # TQM - Мануальні
    ├── crystal-tq.html    # Crystal TQ - Скляні
    ├── tourlock-180.html  # Tourlock 180 - Високобезпекові
    ├── tourlock-120a.html # Tourlock 120A - Безпекові
    └── tourlock-120s.html # Tourlock 120S - Односторонні

```

## Продукція на сайті

### Автоматичні револьверні двері
1. **TQA** - Автоматична двері для низького трафіку
2. **Tournex** - Високопродуктивні двері (до 7400мм діаметр)
3. **Duotour** - Мультифункціональні (обертання/розсування/розпах)

### Мануальні револьверні двері
4. **BoonAssist TQ** - З підсиленням обертання
5. **TQM** - Класичні мануальні двері
6. **Crystal TQ** - Повністю скляні двері

### Безпекові револьверні двері
7. **Tourlock 180** - Високобезпекові двері з технологією StereoVision®
8. **Tourlock 120A** - Безпекові двері для обмеженого простору
9. **Tourlock 120S** - Односторонні безпекові двері (тільки вихід)

## Встановлення на сервер

1. Завантажте всю структуру папок на ваш сервер
2. Переконайтеся, що index.html знаходиться в кореневій директорії
3. Замініть placeholder зображення на реальні фото продукції
4. Налаштуйте форму зворотного зв'язку (потребує серверної обробки)

### Рекомендації щодо зображень
- Формат: JPG або PNG
- Розмір для карток продуктів: 400x300px
- Розмір для великих зображень продуктів: 800x600px
- Оптимізуйте зображення для веб (якість 80-85%)
